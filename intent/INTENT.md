# Doc-Smith Skills Intent

> Claude Code Skills 集合，提供文档生成、图片生成、多语言翻译等能力

状态: draft
最后更新: 2026-02-05

## 愿景

通过 Claude Code Skills 为用户提供一站式的文档和设计资产生成能力：
- 从代码/数据自动生成结构化文档
- AI 生成技术图表和品牌 Logo
- 多语言本地化支持

## Skills 索引

| Skill | 职责 | 状态 |

## 非目标

- 不做通用 AI 对话功能
- 不做代码生成（专注文档和视觉资产）
- 不做实时协作编辑

## 约束

- 依赖 AIGNE CLI 进行图片生成
- Prompts 使用中文编写
- 遵循 Claude Code Plugin 架构规范
